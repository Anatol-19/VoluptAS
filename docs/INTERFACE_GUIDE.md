## 🖥️ Вкладки (Tabs)

### 1. 📊 Таблица (Ctrl+1)

**Основной экран для работы с элементами.**

**Компоненты:**
- **Левая панель:** Дерево иерархии (Module → Epic → Feature → Story)
- **Центр:** Таблица элементов (13 колонок)
- **Правая панель:** Мини-граф связей

**Колонки таблицы:**
| № | Колонка | Inline edit | Описание |
|---|---------|-------------|----------|
| 0 | FuncID | ❌ | Уникальный ID (авто) |
| 1 | Alias | ✅ | Короткий алиас |
| 2 | Title | ✅ | Название |
| 3 | Type | ❌ | Тип элемента |
| 4 | Module | ❌ | Родитель Module |
| 5 | Epic | ❌ | Родитель Epic |
| 6 | Feature | ❌ | Родитель Feature |
| 7 | QA | ❌ | Responsible QA |
| 8 | Dev | ❌ | Responsible Dev |
| 9 | Segment | ✅ | Сегмент (UI, Backend...) |
| 10 | Crit | ✅ | Критичный (✓/пусто) |
| 11 | Focus | ✅ | Фокусный (✓/пусто) |
| 12 | Actions | ❌ | Кнопки действий |

**Фильтры:**
- Поиск по FuncID/Title
- Filter by Type (All, Module, Epic...)
- Filter by Module
- Filter by Epic
- Quick filters: Критичные, Фокусные, Мои

---

### 2. 🌐 Граф (Ctrl+2)

**Визуализация связей между элементами.**

**Режимы:**
- **Иерархия:** Module → Epic → Feature → Story
- **Сегменты:** UI, UX/CX, API, Backend...
- **Полный граф:** Все связи (N:M)

**Функции:**
- Клик по ноде → переход к элементу
- Фильтр по глубине
- Экспорт в PNG

---

### 3. 🧑‍💻 BDD (Ctrl+3)

**Управление BDD сценариями.**

**Возможности:**
- Генерация .feature файлов из FunctionalItem
- Редактирование Gherkin сценариев
- Экспорт .feature файлов

---

### 4. 📋 Трассировки (Ctrl+4)

**Матрица покрытия тестами.**

**Колонки:**
- FunctionalItem
- Test Cases (из Qase)
- Automation Status
- Documentation Links
- GAP (покрытие)

---

### 5. 🏗️ INFRA (Ctrl+5)

**Инфраструктурный слой.**

**Отображение:**
- Service (интеграции, микросервисы)
- Container (Docker, K8s)
- Database (PostgreSQL, MongoDB)
- Subsystems involved
- External services

## 🚀 Стартовый помощник (Starter Wizard)

**Вызов:** `Инструменты → 🚀 Помощник по наполнению базы` (Ctrl+Shift+S)

**Шаблоны:**

### 1. 🧪 QA Management

```
Module: QA Core
└─ Epic: Test Planning
   └─ Feature: Test Cases, Test Plans
└─ Epic: Test Execution
   └─ Feature: Manual Testing, Bug Tracking
└─ Epic: Automation
   └─ Feature: Test Scripts, CI/CD Integration
```

### 2. 📦 Product Development

```
Module: Product
└─ Epic: Features
   └─ Feature: CRUD Operations, Validation
└─ Epic: Bugs
   └─ Feature: Bug Tracking, Hot Fixes
```

### 3. 📘 VoluptAS Documentation (фрактал)

```
Module: VOLUPTAS CORE
└─ Epic: Управление функционалом
   └─ Feature: CRUD элементов, Декомпозиция
└─ Epic: Матрица покрытия
   └─ Feature: Тест-кейсы, Автотесты, Документация
└─ Epic: RACI матрица
   └─ Feature: Назначение ответственных
└─ Epic: Граф связей
   └─ Feature: Визуализация, Фильтрация
```

### 4. 📄 Начать с чистого листа

Пропустить wizard, создать вручную.

## 📝 Inline редактирование

**Двойной клик на ячейку** для редактирования:

| Поле | Редактор | Сохранение |
|------|----------|------------|
| **Alias** | QLineEdit | Enter |
| **Title** | QLineEdit | Enter |
| **Segment** | QLineEdit | Enter |
| **isCrit** | ✓ / пусто | Enter |
| **isFocus** | ✓ / пусто | Enter |

**Валидация:**
- Title не может быть пустым
- isCrit/isFocus: ✓ = 1, пусто = 0

---

## 🎨 Цветовое кодирование

**В таблице:**
- **Критичные (isCrit=1):** 🔴 Красная иконка
- **Фокусные (isFocus=1):** 🟡 Жёлтая иконка

**В графе:**
- **Module:** 🔵 Синий
- **Epic:** 🟢 Зелёный
- **Feature:** 🟠 Оранжевый
- **Story:** 🟣 Фиолетовый
- **Service:** 🔴 Красный
